Sourcedown

It's ill-Literate programming for the masses!

Will take your source hierarchy and generate a bunch of HTML files. This takes the Docco approach and starts to open it up to new languages.

Running Sourcedown

This is my typical run: 

java -jar sourcedown.jar -s src/main/scala -o docs

This will convert all files under the src/main/scala into HTML files underneath the docs folder. This will preserve files, so the first file you'll see will be in the format src/main/scala/{package}/{file}.html.

Note that I'm typically running this within the root of the project I'm working on. And, that sourcedown.jar can be wherever.
